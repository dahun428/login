액션태그란

JSP페이지 내에서 어떤 동작을 하도록 지시하는 태그 입니다.
예를 들어, 페이지 이동, 페이지 include 등 입니다.
추후에 배울 bean과 관련된 태그도 있습니다.
우선은 forward, include, param 태그만 살펴보고 , 추후 bean 을 학습 할 때 추가로 학습하도록 하겠습니다.

forward, include, param 태그 살펴보기

  1. forward

    현재 페이지에서 다른 특정페이지로 전환할 때 사용 합니다. 사용방법이 간단!

    	jsp:forward page = "sub.jsp"
  2. include

    현재 페이지에 다른 페이지를 삽입할 때 사용한다. 

     include01.jsp 페이지 입니다.
 jps:include page ="include02.jsp" flush = "true"
 다시 include01.jsp 페이지 입니다.
  3. param

    forward 및 include 태그에 데이터 전달을 목적으로 사용되는 태그. 이름과 값으로 이루어져있습니다. 

    jsp:forward page = "forward_param.jsp"
	jsp:param name = "id" value = " acdef"
	jsp:param name = "pw" value = "1234"
/jsp:forward
